Detailed Description

Precision::General_Base::Dynamic::Abstract::Int provides an image-less precision integer with basic integer operations from arithmetic to sign manipulation. This class will also serve as the core to all dynamic (i.e. mutable radix) number types in the Precision namespace. Results of operations involving multiple integers will share the base of the left hand side value.
