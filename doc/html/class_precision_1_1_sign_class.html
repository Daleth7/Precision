Detailed Description

Precision::SignClass provides a representation for a number sign such that all number types can store a number's magnitude and numerical sign separately. This class also simplifies sign related operations and follows these rules:

  • Negative == -1
  • Positive == +1 For purposes of conversion, positive also includes [0, inf).
